Get youself a partner

Process & Special Feature

For the last project, I created a small memory game. There are 6 pairs of card and have been randomized. User need to pair them as fast as possible. My special feature for this project is mostly about UI. I add a step count which will count how many steps the user take to finish the game, also there is a timer, which will count how long it takes to win the game. After the user win the game, there will be a summary screen that have all the infomation display as well as a play-again button.
